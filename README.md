# Employee Management System

This is a simple **Employee Management System** built using **Java Swing** for the graphical user interface (GUI) and **JDBC** for database connectivity. It allows users to manage employees by adding, updating, deleting, and viewing employee records.

## Features

- **Add Employees**: Enter employee details such as name, ID, designation, department, and salary.
- **View Employees**: Display a list of all employees.
- **Update Employee**: Edit employee details by searching for an employee ID.
- **Delete Employee**: Remove employee records from the system.

## Technology Stack

- **Java Swing**: GUI framework for building the user interface.
- **JDBC**: For connecting to the database (MySQL/SQLite or other DBMS).
- **MySQL** (or other database): Backend database to store employee records.

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Java Development Kit (JDK) version 8 or later
- MySQL or SQLite for database (optional: you can use another database if preferred)
- An IDE such as **IntelliJ IDEA**, **Eclipse**, or **NetBeans**
- JDBC driver for the database you're using (e.g., MySQL Connector/J for MySQL)
  
## Usage
Once the application is running, the main menu will appear with options to:
- Add a new employee
- View all employees
- Update employee details
- Delete an employee
You can manage employee records by navigating through these options.
